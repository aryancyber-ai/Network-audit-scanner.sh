# Network-audit-scanner.sh

A Bash-based cybersecurity and system administration tool that collects important Linux system information for security auditing and troubleshooting.

## Features

- System Information
- Hostname Details
- Logged-In Users
- Open Ports
- Network Information
- Firewall Status
- Disk Usage
- Memory Usage
- Running Services

## Technologies Used

- Linux
- Bash Scripting
- Networking
- Cybersecurity Fundamentals

## Usage

```bash
chmod +x system_audit.sh
./system_audit.sh
```

## Learning Objectives

This project demonstrates:

- Linux Administration
- Bash Scripting
- Security Auditing
- System Monitoring
- Network Enumeration
